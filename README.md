# AppIconTag

自用App Icon修改命令行工具

```shell
/*
 * AppIconTag [-fsctgh] SourcFile
 * -s  : size, image width and height
 * -c  : text color
 * -bc : text label backgroundColor
 * -t  : text
 * -f  : fontSize
 * -g  : gravity Top/Bottom
 * -h  : text label height
 */
 
AppIconTag ICON_PATH -f 28 -s 180 -c 0x0000FFFF -bc 0xFF00FFFF -t "TEXT" -g Top -h 40
```
